# 开发环境搭建

## 下载 JDK

### 推荐使用：JetBrains Runtime (JBR)

- [JetBrains Runtime](https://github.com/JetBrains/JetBrainsRuntime/releases)
- Windows 平台用户可以下载 `jbr_jcef-25.x.x-windows-x64-xxxx.zip`
- 下载后解压至任意路径下即可，此处以 `C:\Program Files\JetBrains\JetBrainsRuntime-25` 为例
- 在 `CMD` 中运行以下命令将 JBR 添加到系统 PATH 环境变量中
   ```cmd
   setx JAVA_HOME "C:\Program Files\JetBrains\JetBrainsRuntime-25"
   setx PATH "%JAVA_HOME%\bin;%PATH%"
   ```

### 为什么推荐 JetBrains Runtime？

> JetBrains Runtime 是一个适用于 Windows、Mac OS X 和 Linux 的 OpenJDK 分支，专门为现代桌面应用和开发工具优化。相比标准OpenJDK，它提供了以下优势：

🚀 核心优势：

1. 增强的类重定义（DCEVM）
    - 支持热插拔，无需重启即可重新加载修改的类
    - 极大提升开发调试效率
2. 改进的桌面集成
    - 增强的字体渲染：更好的文本清晰度和抗锯齿效果
    - HiDPI支持：在高分辨率显示器上完美显示
    - 键盘支持：更好的键盘事件处理和快捷键支持
    - 窗口/焦点子系统：改进的窗口管理和焦点处理
3. 性能优化
    - 针对IDE使用场景优化
    - 更快的启动时间
    - 改进的内存管理
4. 增强功能
    - 可选JCEF：内置Chromium嵌入式浏览器框架
    - 无障碍功能：更好的辅助技术支持
    - 提前修复：包含OpenJDK中尚未发布的bug修复
